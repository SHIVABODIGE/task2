# Exploratory Data Analysis & Business Intelligence

## Overview

This project performs Exploratory Data Analysis (EDA) on a sales dataset using Python and MongoDB. The goal is to identify business trends, customer behavior patterns, and revenue drivers through statistical analysis, visualizations, and aggregation queries.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* MongoDB

## Project Highlights

* Total Revenue: ₹139.4 Million
* Total Customers: 947
* Top Category: Electronics
* Top Product: Laptop
* Top City: Patna
